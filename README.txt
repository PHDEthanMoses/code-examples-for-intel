Данные файлы содержат примеры моего 
кода на python 3 для анализа данных, 
машинного обучения и Deep Learning. 
Описание:

1) Random forests - решение задачи кредитного скоринга 
при помощи RandomForestClassifier;

2) Budget risk - Лабороторная работа 
по муниципальным финансам. При помощи линейной регрессии 
изучаем зависимость между коэфф. собираемости бюджета и
бюджетным риском;

3)MNIST - классическая задача 
распознования рукописных цифр, решенная при помощи 
стохастическго градиентного спуска;

4)FNN classification - определение сорта 
вина при помощи нейронной сети прямого распространения

5)House prise - стандартная задача 
регрессии о прогнозировании цены на дом.

6) Clothesclassifier - исследование 
fashion MNIST dataset. Распознование осуществлялось 
при помощи FFNN.
